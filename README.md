# Advanced Data Visualization & Storytelling Dashboard

Bu proje, Python ve Matplotlib kullanarak veriyi bir hikaye akışıyla sunan interaktif görünümlü bir dashboard çalışmasıdır.

## Özellikler
- **Aylık Satış Trendi:** Zaman serisi analizi.
- **Ürün Karşılaştırması:** Kategorik performans ölçümü.
- **Bölgesel Analiz:** Mekansal satış dağılımı.
- **Müşteri Harcamaları:** İstatistiksel dağılım (Histogram).

## Kullanılan Teknolojiler
- Python
- Pandas & Numpy
- Matplotlib (Subplots, Annotations)
